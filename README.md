## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/TechGeeks-Club/Trailer-secript.git
    ```

2. Navigate to the project directory:

    ```bash
    cd Trailer-secript
    ```

3. Install dependencies:

    ```bash
    pip install -r requirements.txt
    ```

4. Run it:

    ```bash
    python3 Trailer.py
    ```
